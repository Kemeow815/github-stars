# 个人博客

基于 Astro 构建的极简个人博客。

## 技术栈

- **Astro** - 静态网站生成器
- **UnoCSS** - 原子化 CSS (这个暂时没有整合好，后续考虑进一步整合或替换)
- **Expressive Code** - 代码语法高亮

## 开发

```bash
bun dev                # 启动开发服务器
bun build              # 构建生产版本
bun run lint           # 调用 autocorrect 格式化内容
bun run subset-font    # 提取字体子集
```
